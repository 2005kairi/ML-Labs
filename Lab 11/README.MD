This project analyzes a loan dataset to predict whether a borrower will fully repay their loan. After loading and exploring the data, visualizations were created to examine relationships between FICO scores, interest rates, loan purposes, and repayment outcomes. Categorical features were converted using dummy variables, and the data was split into training and testing sets. Two machine learning models were built: a Decision Tree and a Random Forest classifier. Their performance was evaluated using classification reports and confusion matrices. Overall, the Random Forest achieved higher accuracy, while the Decision Tree identified more high‑risk borrowers.
